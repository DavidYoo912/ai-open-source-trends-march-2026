OpenClaw started as an app. It's now behaving like an OS.
Four signals from 756 open-source AI repos tracked in this snapshot.

OpenClaw is a self-hosted AI assistant that runs on your own machine — and 26 independent projects on GitHub now exist specifically because it does. Here's what else the data showed.

Story 2
"Skills" are the new plugins — a new layer of the AI stack is forming
38 repos in this index use "skill" to describe reusable AI capabilities you can install into agents — like plugins, but for behaviour rather than features. One library lists 5,400 curated skills. Another has 1,304 installable skills for coding agents. This is a new abstraction layer that didn't exist in meaningful form 12 months ago: not models, not apps, but shareable bundles of instructions that make agents smarter at specific tasks.

Story 3
AI agent security has become its own category of tooling
47 of the 61 security repos in this snapshot are currently active — not traditional antivirus, but a new category built specifically for AI agents: firewalls that intercept what agents are about to do, sandboxes that isolate them, secret-leak scanners, prompt injection blockers. Security for AI agents appears to be consolidating into a distinct layer of the stack, separate from the agents themselves.

Story 4
33,550 developers forked a repo that leaks AI coding tools' internal instructions
x1xhlol/system-prompts-and-models-of-ai-tools contains the leaked system prompts of Cursor, Devin, Replit, Lovable, Windsurf, and 20+ other AI coding tools. It has 132k stars. More tellingly, it has 33,550 forks — meaning tens of thousands of developers didn't just bookmark it, they copied it. They're studying how these products are built, either to learn from them or to build against them.
